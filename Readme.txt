┌──────────────────────────────────────────────────────────┐
│ Обозреватель макросов + скрипт для интеграции в AkelPad. │
│ Версия от 27 января 2013.                                │
└──────────────────────────────────────────────────────────┘

Софтинка сканирует штатные макросы и выводит их в виде списка. В целом сами разберетесь.
Поддерживаемые параметры командной строки:
-Title=<заголовок>   — изменить заголовок окна
-GoTo=<макрос>       — при открытии перейти к указанному макросу
-OpenViewer=<режим>  — обработка даблклика: Disabled - ничего не делать
                                            Enabled - открыть в установленном текстовом редакторе (по умолчанию)
                                            Return - вывести данные в консоль и закрыть программу (для интеграции)

Скрипт по дефолту ищет экзешник <AkelPad>\AkelFiles\Tools\Macro Browser.exe
Если не хотите его туда пихать, измените переменную ProgramPath.

Требования: Винда, Веснот, .NET Framework 2.0
